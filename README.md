## Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

### Deployment
The site is hosted on GitHub Pages. Any changes pushed to the `main` branch will automatically update the live site.

## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary: #6f7cff;      /* Primary color */
    --bg: #0a0a1e;           /* Background */
    --text: #fff;            /* Text color */
    --muted: #b0b8cc;        /* Muted text */
}
```

### Content
Edit `index.html` to change:
- Hero section text and images
- Feature list items
- Form fields and labels
- Navigation menu links

### Images
Replace image URLs in `index.html`:
- Logo: Update `src` in `.logo img`
- Hero background: Update `background-image` in `.hero`
- Hero banner: Update `src` in `.hero-image-wrapper img`
